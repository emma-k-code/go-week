# API 連線時間測試 (使用併發進行連線)

### 使用方法
```
go run main.go -m=[連線方法] -p=[API路徑] -d=[API參數] -n=[連線次數]

-m: 必填，GET/POST/PUT/DETELE (預設為 GET)，不用區分大小寫
-p: 必填
-d: 非必填，資料格式 key1=val2&key2=val2
-n: 非必填 (預設為 5)
```

### 使用範例
```
go run main.go -p='http://weather.json.tw/api' -n=10
```
